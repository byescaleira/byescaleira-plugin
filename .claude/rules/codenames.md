## Branch naming

```
feature/<codename>-<short-description>
fix/<codename>-<short-description>
spike/<codename>-<short-description>
release/<version>-<planet>
```

## How to choose a codename

1. Pick the category (project, module, release, environment).
2. Match the intended meaning.
3. If conflicted, prefer shorter names.
4. Avoid names that sound like Apple frameworks (SwiftUI, Combine, SwiftData).
