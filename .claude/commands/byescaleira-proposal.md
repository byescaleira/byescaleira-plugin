# /byescaleira-proposal

Create a `PROPOSAL.md` for a new byescaleira project.

## Steps

1. Ask for the project codename (or pick one together from the byescaleira space dictionary).
2. Ask the 7 birth questions:
   - What real problem does this solve?
   - Who is it for?
   - How will we know it worked in 3 months?
   - What is the MINIMUM useful version?
   - What is OUT of initial scope?
   - Why is Rafael the right person to build this?
   - If this fails, what is the smallest price paid?
3. Write a one-page `PROPOSAL.md` in English, using the byescaleira voice.
4. Keep the tone direct and practical. No hype.
5. End with a Decision table and Open Questions checklist.

## Output

A `PROPOSAL.md` file ready for approval before any code is written.
